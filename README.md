# Python Basics Assignment

This repository contains beginner-level Python exercises 
implemented using Jupyter Notebook.

## 📌 Topics Covered

- Variables and Data Types
- Arithmetic and Logical Operators
- Conditional Statements (if, elif, else)
- Loops (for, while)
- String methods and String Slicing
- Python Functions
- Positional Arguments
- Scope of Variables (Local and Global)
- Basic problem-solving exercises

## 🎯 Objective

The objective of this assignment is to build a strong 
foundation in Python programming, understand conditional 
logic, function behavior, and variable scope, and improve 
logical thinking skills.

## 🛠️ Tools Used

- Python 3
- Jupyter Notebook
